Shared B Matrix
SensorEncoder 和 QueryEncoder 共用同一個 RFF 隨機矩陣 B（register_buffer），確保座標編碼空間一致，使 cross-attention 的 Q 和 KV 在相同特徵空間對齊。
101 KV Tokens
Ns=100 感測器 token + 1 個 Re token（prepend），共 101 個 KV token 進入 cross-attention。Re token 攜帶 Reynolds number 資訊，使模型條件化於流場參數。
ComponentScaler
u、v、p 的數量級差異大（壓力 ~O(1)，速度 ~O(0.01)），ComponentScaler 為每個分量學習獨立的 scale 和 bias，初始化為 ones/zeros，訓練中自動調整。

05   Key Code
src/pi_onet/pit_ldc.py — rff_encode & CrossAttentionOperator.forward
1def rff_encode(z: Tensor, B: Tensor) -> Tensor:
2 # Random Fourier Features: maps [N, 2] → [N, 2*rff_features]
3 # B: [2, D] — shared between SensorEncoder and QueryEncoder
4 proj = 2 * torch.pi * z @ B # [N, D]
5 return torch.cat([proj.sin(), proj.cos()], dim=-1) # [N, 2D]
6
7class CrossAttentionOperator(nn.Module):
8 def forward(self, sensors: Tensor, xy: Tensor,
9 c: Tensor, re_norm: float) -> Tensor:
10 # ── Sensor branch (Key / Value) ──────────────────────
11 sens_xy = sensors[:, :2] # [Ns, 2]
12 sens_uvp = sensors[:, 2:] # [Ns, 3]
13 rff_s = rff_encode(sens_xy, self.B) # shared B
14 kv_tok = self.sensor_encoder(rff_s, sens_uvp, re_norm)
15 # kv_tok: [Ns+1, d] (re_token prepended inside SensorEncoder)
16 # ── Query branch ─────────────────────────────────────
17 rff_q = rff_encode(xy, self.B) # same B → aligned feature space
18 cemb = self.comp_embed(c) # Embedding(c, 8)
19 q_tok = self.query_proj(torch.cat([rff_q, cemb], -1)) # [Nq, d]
20 # ── Cross-Attention ──────────────────────────────────
21 out = self.cross_attn(
22 query=q_tok, key=kv_tok, value=kv_tok # Q from query, K/V from sensors
23 )[0] # [Nq, d]
24 out = self.out_proj(self.norm(out)) # [Nq, 1]
25 # ── ComponentScaler ──────────────────────────────────
26 s = self.comp_scale[c].unsqueeze(-1) # [Nq, 1]
27 b = self.comp_bias[c].unsqueeze(-1)
28 return out * s + b # per-component scale/bias
