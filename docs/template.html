6def rff_encode(z: Tensor, B: Tensor) -> Tensor:


7    """Random Fourier Features positional encoding.


8    


9    Maps spatial coords through a frozen random frequency matrix


10    to approximate shift-invariant kernels (Bochner's theorem).


11    Shared between SensorEncoder and QueryEncoder to align frequency


12    basis across both input streams.


13    


14    Args:


15        z: [N, 2]  spatial coordinates (x, y) ∈ [0, 1]²


16        B: [2, F]  frozen frequency matrix (register_buffer)


17    Returns:


18        Tensor of shape [N, 2F]  (sin and cos features concatenated)


19    """


20    proj = 2.0 * math.pi * z @ B         # [N, F]


21    return torch.cat([proj.sin(), proj.cos()], dim=-1)  # [N, 2F]


22


23


24class CrossAttentionOperator(nn.Module):


25    def forward(


26        self,


27        sensors: Tensor,     # [N_s, 5]  raw sensor readings


28        re_norm: float,      # (Re − 4000) / 816


29        xy:      Tensor,     # [N_q, 2]  query coordinates


30        c:       Tensor,     # [N_q]     component index  0=u 1=v 2=p


31    ) -> Tensor:            # [N_q, 1]


32        kv  = self.sensor_encoder(sensors, re_norm)  # [N_s+1, d]


33        q   = self.query_encoder(xy, c)             # [N_q,   d]


34        out = self.cross_attn(q, kv, kv)[0]         # [N_q,   d]


35        out = self.proj(self.norm(out))              # [N_q,   1]


36        return out * self.component_scale[c].unsqueeze(-1) \


37                   + self.component_bias[c].unsqueeze(-1)

Shared B Matrix
SensorEncoder 和 QueryEncoder 共用同一個 RFF 矩陣，確保兩者的座標編碼在相同特徵空間，使 cross-attention Q/KV 可以正確對齊。
101 KV Tokens
N_s=100 sensor tokens + 1 個 Re token (prepended)。Re token 攜帶 Reynolds number，讓模型條件化於流場參數。
14   Sensor Domain Map
Sensor Placement — LDC [0,1]²
Collocation Points — Physics Sampling
Sensor Sampling
Interior sensors sampled uniformly at random from [0.05, 0.95]². Boundary sensors fixed at 5 equally-spaced pts per wall. Same sensor set reused across all training steps.
Physics Collocation
Collocation points resampled each step from Uniform([0,1]²). Forces the physics residuals (NS equations) to be evaluated at diverse locations, preventing overfitting to a fixed grid.
